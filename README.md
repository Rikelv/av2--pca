
# AV2 - PCA  
## Sistema simples de cadastro de alunos (CRUD) em Python

Este repositório contém o trabalho da AV2 da disciplina PCA.

### Arquivos:
- `main.py` → programa principal (CRUD)
- `alunos.csv` → gerado automaticamente
- `AV2_PCA_Rikelv.txt` → arquivo para envio no AVA

Execute com:

```
python main.py
```
